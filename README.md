Here, we present the Python code for the following publication:

R. Aryapratama, S. Schulte, S. Pauliuk, Climate change mitigation potential of producing wood-based materials and energy from restoring degraded land in Indonesia – A nation-wide scenario analysis, _Biomass and Bioenergy_ 211 (2026) 109191. https://doi.org/10.1016/j.biombioe.2026.109191.

Note:
The .py files in this GitHub contains the Python scripts to run the main Degraded Land Scenarios (DLS1, DLS2, and DLS3) in our paper.
